# Task 15 - Customer Segmentation (RFM Analysis)

This project performs RFM (Recency, Frequency, Monetary) analysis
on the Online Retail II dataset.

## Steps Performed
- Data cleaning (removed nulls & canceled orders)
- Feature engineering (TotalPrice)
- RFM calculation
- Quantile-based scoring
- Customer segmentation
- Visualization of segments
- Business action recommendations

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Google Colab

## Output Files
- rfm_segments.csv
- segment_actions.txt

## Outcome
Identified Champions, Loyal Customers, Potential customers, and At Risk customers
for targeted marketing strategies.
